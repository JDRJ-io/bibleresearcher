# 2) Add the hologram animation style

Append this to your `animated-backgrounds.css` (from the previous message).
It’s smooth, low‑CPU, honors reduced‑motion, and uses **conic + linear gradients** with a gentle hue rotation for that iridescent feel.

```css
/* ==============================
   HOLOGRAPHIC BACKGROUND
   ============================== */

.bg-hologram {
  /* multi-layered sheen */
  background:
    /* soft base */
    radial-gradient(120% 80% at 50% 100%, color-mix(in oklab, var(--bg-primary), #000 5%) 0%, transparent 70%),
    /* iridescent film */
    conic-gradient(from 180deg at 50% 50%,
      color-mix(in oklab, var(--holo-a), transparent calc((1 - var(--holo-sheen)) * 100%)) 0deg,
      color-mix(in oklab, var(--holo-b), transparent calc((1 - var(--holo-sheen)) * 100%)) 90deg,
      color-mix(in oklab, var(--holo-c), transparent calc((1 - var(--holo-sheen)) * 100%)) 180deg,
      color-mix(in oklab, var(--holo-d), transparent calc((1 - var(--holo-sheen)) * 100%)) 270deg,
      color-mix(in oklab, var(--holo-a), transparent calc((1 - var(--holo-sheen)) * 100%)) 360deg
    ),
    /* soft glare sweeps */
    linear-gradient(120deg,
      color-mix(in oklab, var(--holo-a), transparent 85%) 0%,
      transparent 40%,
      color-mix(in oklab, var(--holo-b), transparent 85%) 60%,
      transparent 100%
    ),
    var(--bg-primary);
  background-blend-mode: screen, screen, overlay, normal;
  filter: saturate(1.05);
  animation: holo-shift var(--anim-speed, 26s) ease-in-out infinite alternate;
}

@keyframes holo-shift {
  0% {
    transform: translateZ(0); /* keep on GPU */
    background-position: 50% 100%, 50% 50%, 0% 0%, 0 0;
    filter: hue-rotate(0deg) saturate(1.05) brightness(1);
  }
  50% {
    background-position: 52% 98%, 48% 52%, 10% 0%, 0 0;
    filter: hue-rotate(12deg) saturate(1.08) brightness(1.03);
  }
  100% {
    background-position: 48% 102%, 52% 48%, -8% 0%, 0 0;
    filter: hue-rotate(-8deg) saturate(1.06) brightness(1.02);
  }
}
```

*(The rest of the animations—`bg-aurora`, `bg-bokeh`, `bg-stars`, `bg-waves`, `bg-gradient-shift`—are exactly as in my last message. Keep those in `animated-backgrounds.css`.)*

# 4) How to test quickly

1. Toggle your `<html>` class to a new theme ID:

```js
document.documentElement.className = 'hologram'; // or 'eden', 'royal', etc.
```

2. Ensure your ThemeManager also accepts these IDs and sets the same class on `<html>` (and/or applies the variables programmatically).

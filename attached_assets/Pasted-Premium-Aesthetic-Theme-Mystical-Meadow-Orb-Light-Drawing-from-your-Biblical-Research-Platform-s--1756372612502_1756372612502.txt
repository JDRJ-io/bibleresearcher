Key Design Choices
	•	Inspiration: Enchanted meadow with grass swaying, floating orbs/lights (like fireflies in daylight), soft glows for a mystical, uplifting vibe—perfect for light-mode research without eye strain.
	•	Colors: Fresh greens and soft blues for the meadow; warm yellow-golds for orbs/lights. High-contrast text (e.g., dark greens on light backgrounds) for WCAG compliance.
	•	Animations: Video for ambient meadow movement; CSS particle system for floating orbs (lightweight, randomized paths for natural feel).
	•	Performance: Video loops seamlessly (low-res fallback for mobile); animations use transform/opacity for GPU efficiency.
	•	Accessibility: Contrasts >7:1; media query for reduced motion (static orbs); no flashing to avoid seizures.
	•	Implementation: Add to index.css and app root. Host video in Replit (e.g., /public/video/meadow-fireflies.mp4).

.orb-overlay {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  z-index: -1; /* Over video, under content */
  pointer-events: none;
  overflow: hidden;
}

.orb {
  position: absolute;
  width: 20px; /* Vary size */
  height: 20px;
  border-radius: 50%;
  background: radial-gradient(circle, rgba(255, 255, 200, 0.8), rgba(255, 255, 100, 0.2)); /* Glowing yellow orb */
  box-shadow: 0 0 10px rgba(255, 255, 100, 0.6);
  animation: float 15s infinite ease-in-out;
  opacity: 0.7;
}

/* Randomize positions and animations via inline styles or JS, e.g., left: random%, top: random%, animation-delay: -random s */
@keyframes float {
  0% { transform: translateY(0) translateX(0); }
  50% { transform: translateY(-20vh) translateX(5vw); } /* Gentle drift */
  100% { transform: translateY(0) translateX(0); }
}

/* Reduce motion */
@media (prefers-reduced-motion: reduce) {
  .orb {
    animation: none;
    opacity: 0.5; /* Static glow */
  }
}
For more dynamism, use JS to randomize orb sizes/delays (e.g., in React: style={{ left: ${Math.random()*100}%, animationDelay: -${Math.random()*15}s }}).
